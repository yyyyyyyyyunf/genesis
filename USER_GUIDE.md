# Genesis 用户使用指南

欢迎使用 Genesis 平台！本指南将帮助你快速上手，从零开始创建精美的营销页面。

## 📖 Genesis 平台简介

Genesis 是一个现代化的低代码营销页面搭建平台，通过可视化编辑器和 AI 助手，让你无需编写代码即可快速创建和发布页面。

**核心特点**:
- 🎨 **可视化编辑**: 拖拽组件，所见即所得
- 🤖 **AI 辅助**: 通过自然语言描述，让 AI 帮你生成页面配置
- ⚡ **实时预览**: 编辑即时生效，立即看到结果
- 📱 **响应式设计**: 自动适配不同设备屏幕

---

打开浏览器，访问 Zeus 编辑器：
- **本地开发**: http://localhost:3000
- **线上环境**: 你的部署地址

## 🖥️ 界面布局介绍

Zeus 编辑器由四个主要区域组成：

<!-- 截图：标注四个区域的界面图 -->

### 1. 左侧 - 楼层树 (Floor Tree)
显示页面的组件层级结构，支持拖拽排序和嵌套。

### 2. 中间 - 实时预览 (Preview)
iframe 内嵌 Hercules 渲染引擎，实时显示页面效果。

### 3. 右侧上方 - 属性面板 (Property Inspector)
选中组件后，显示该组件的可编辑属性，基于 Schema 自动生成表单。

### 4. 右侧下方 - AI 助手 (Chat Panel)
与 AI 对话，通过自然语言指令修改页面配置。

## 📝 使用 Zeus 编辑器

### 楼层树操作

#### 添加组件

1. 点击楼层树顶部的 **"+ 添加楼层"** 按钮
2. 从下拉菜单中选择组件类型（如 Text、Image、Button 等）
3. 新组件将添加到页面底部

<!-- 截图：添加组件的下拉菜单 -->

**提示**: 每个组件在楼层树中称为一个"楼层"（Floor）。

#### 删除组件

1. 在楼层树中找到要删除的组件
2. 点击组件右侧的 **删除图标** (🗑️)
3. 确认删除

<!-- 截图：楼层树中的删除按钮位置 -->

#### 调整顺序

Genesis 使用拖拽排序功能，让你轻松调整组件顺序：

1. 鼠标悬停在楼层树中的组件上
2. 按住鼠标左键，拖动组件到目标位置
3. 松开鼠标完成排序

<!-- 截图：拖拽排序的动作示意 -->

**提示**: 拖拽时会显示蓝色的放置指示线。

#### 嵌套组件 (容器)

部分组件支持嵌套子组件，例如：
- **Tab**: 每个 Tab 页可以包含多个子组件
- **Accordion**: 每个手风琴项可以包含多个子组件
- **Shelf**: 横向排列多个子组件

**操作方法**:
1. 将子组件拖拽到容器组件的内部
2. 缩进层级表示嵌套关系

<!-- 截图：Tab 组件的嵌套结构示例 -->

---

### 属性面板使用

选中楼层树中的任意组件后，右侧属性面板会自动显示该组件的可编辑属性。

<!-- 截图：属性面板的完整界面 -->

#### 常见属性类型

1. **文本输入框**: 用于输入文字内容、URL 等
   - 示例：Text 组件的 `content` 字段

2. **数字输入框**: 用于输入数值，部分带单位（如 `px`）
   - 示例：Image 组件的 `height` 字段

3. **下拉选择**: 从预定义选项中选择
   - 示例：Text 组件的 `align` 字段（左对齐/居中/右对齐）

4. **颜色选择器**: 选择颜色值
   - 示例：Text 组件的 `color` 字段

5. **开关按钮**: 布尔值（是/否）
   - 示例：Video 组件的 `autoplay` 字段

<!-- 截图：不同属性类型的示例 -->

#### 实时更新

属性面板中的任何修改都会 **立即反映** 到中间的预览区域，无需点击保存按钮。

---

### 实时预览

中间的预览区域通过 iframe 加载 Hercules 渲染引擎，实时显示页面效果。

**特性**:
- ✅ 自动同步：编辑后立即更新
- ✅ 滚动交互：可以滚动预览整个页面
- ✅ 组件点击：点击预览中的组件可在楼层树中定位

<!-- 截图：预览区域的交互效果 -->

---

### 保存和加载配置

#### 保存配置

1. 点击顶部工具栏的 **"保存"** 按钮
2. 配置将自动保存到后端（Jarvis）
3. 成功后显示提示信息

<!-- 截图：保存按钮的位置 -->

#### 加载配置

- 页面加载时，系统会自动从后端读取最新的配置
- 刷新页面即可重新加载配置

**技术说明**: 配置数据以 JSON 格式存储，包含所有组件的类型和属性。

## 🤖 使用 AI 助手

Genesis 的 AI 助手让你可以通过自然语言来修改页面，无需手动操作楼层树和属性面板。

<!-- 截图：AI 助手面板 -->

### 发送指令

1. 在 AI 助手的输入框中输入你的需求
2. 按 **Enter** 或点击 **"发送"** 按钮

**示例指令**:
```
帮我把 Banner 图片改大一点
```
```
在顶部添加一个标题，内容是"欢迎来到我的店铺"
```
```
把第二个按钮的颜色改成蓝色
```

<!-- 截图：输入指令的界面 -->

### Draft/Commit 工作流

为了安全性，AI 的修改不会立即生效，而是先进入 **Draft（草稿）** 状态，等待你的审查。

<!-- 截图：Draft 状态的提示条 -->

#### 工作流程

1. **AI 生成配置**: AI 根据你的指令生成新的页面配置
2. **预览变更**: 预览区域会显示 AI 建议的修改效果
3. **审查决策**: 你可以选择"接受"或"拒绝"

<!-- 截图：接受/拒绝按钮 -->

#### 接受更改

点击 **"接受"** 按钮，AI 的修改将正式应用到页面配置中。

#### 拒绝更改

点击 **"拒绝"** 按钮，页面将恢复到 AI 修改前的状态。

**提示**: 这个机制确保了 AI 不会破坏你的页面，你始终拥有最终决定权。

## 🎯 常见操作场景

下面是一些典型的页面搭建场景，帮助你快速上手。

### 场景 1: 创建首页 Banner

<!-- 截图：完成后的 Banner 效果 -->

**步骤**:

1. **添加 Image 组件**
   - 点击 "+ 添加楼层" → 选择 `Image`

2. **配置图片属性**
   - `variant`: 选择 `background`（背景图片模式）
   - `src`: 输入图片 URL
   - `height`: 设置为 `400`（高度 400px）
   - `aspectRatio`: 选择 `16/9`

3. **查看效果**
   - 预览区域会显示全屏的背景图片

**AI 助手快捷方式**:
```
帮我添加一个 Banner，使用这个图片：https://example.com/banner.jpg，高度 400px
```

---

### 场景 2: 添加图片轮播

<!-- 截图：Carousel 组件效果 -->

**步骤**:

1. **添加 Carousel 组件**
   - 点击 "+ 添加楼层" → 选择 `Carousel`

2. **配置轮播属性**
   - `images`: 点击添加多张图片，每张图片需要填写：
     - `src`: 图片 URL
     - `alt`: 图片描述
   - `autoplay`: 开启自动播放
   - `interval`: 设置切换间隔（如 `3000` 毫秒）

3. **预览效果**
   - 预览区域会显示自动轮播的图片

---

### 场景 3: 创建产品展示区

<!-- 截图：Shelf 组件的产品展示效果 -->

**步骤**:

1. **添加 Text 组件作为标题**
   - `content`: 输入 "热门产品"
   - `size`: 选择 `2xl`
   - `align`: 选择 `center`

2. **添加 Shelf 组件（横向布局容器）**
   - 点击 "+ 添加楼层" → 选择 `Shelf`

3. **在 Shelf 中添加子组件**
   - 拖拽多个 `Image` 组件到 Shelf 内部
   - 每个 Image 配置不同的产品图片

4. **美化 Shelf**
   - `gap`: 设置间距为 `16`
   - `padding`: 设置内边距为 `24`

**AI 助手快捷方式**:
```
帮我创建一个产品展示区，包含一个标题"热门产品"和三个产品图片横向排列
```

---

### 场景 4: 配置表单

<!-- 截图：Form 组件效果 -->

**步骤**:

1. **添加 Form 组件**
   - 点击 "+ 添加楼层" → 选择 `Form`

2. **配置表单字段**
   - `fields`: 点击添加多个字段，每个字段配置：
     - `name`: 字段标识（如 `email`）
     - `label`: 显示文本（如 "邮箱地址"）
     - `type`: 字段类型（如 `email`）
     - `required`: 是否必填

3. **配置提交按钮**
   - `submitText`: 按钮文字（如 "提交"）
   - `submitUrl`: 提交地址

4. **测试表单**
   - 在预览区域填写表单并提交

## 💡 最佳实践和技巧

### 1. 合理使用容器组件

使用 **Shelf**、**Tab**、**Accordion** 等容器组件来组织内容，让页面结构更清晰。

### 2. 善用 AI 助手

对于复杂的布局调整，使用 AI 助手可以节省大量时间：
```
帮我把所有的标题字体改大，并居中对齐
```

### 3. 定期保存

虽然编辑器支持自动保存，但在做重大修改前，建议手动点击保存按钮。

### 4. 使用有意义的组件顺序

在楼层树中保持清晰的组件顺序，便于后续维护：
- 顶部导航
- Banner
- 内容区块
- 底部信息

### 5. 预览不同设备

定期调整浏览器窗口大小，检查页面在不同屏幕尺寸下的效果。

### 6. 利用 Schema 元数据

属性面板中的：
- **@labels**: 显示中文标签，方便理解
- **@unit**: 显示单位（如 px），避免输入错误
- **@default**: 了解默认值，按需修改
